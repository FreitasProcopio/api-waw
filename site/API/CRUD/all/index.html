    Endpoint /all

    O endpoint /all permite consultar todas as palavras cadastradas no sistema.

    Visão Geral

    • Método: GET
    • URL: /all
    • Descrição: Retorna um array com todas as palavras e seus respectivos campos.

    Exemplo de Requisição

    curl http://localhost:3000/all
